## Color Palette Reference

```
Primary Green:    #00A651  (backgrounds, CTAs)
Navy Blue:        #1E3A8A  (text, accents)
Bright Blue:      #3B82F6  (highlights)
Yellow Accent:    #FCD34D  (links, badges)
White:            #FFFFFF  (text on dark)
Cream/Light:      #F5F5DC  (logo details)
Dark Text:        #1F2937  (on light backgrounds)
```

## Icon Style Guide

- **Documents:** Simple line icons or filled rectangles with fold
- **Checkmarks:** Bold, rounded corners
- **Lock/Security:** Closed padlock, simple outline
- **Trademark:** ¬Æ symbol in circle badge
- **Social:** Standard platform icons (white or yellow on dark)

## Typography

- **Headings:** Bold sans-serif (Arial Black, Montserrat Bold, or similar)
- **Body:** Clean sans-serif (Arial, Helvetica, Open Sans)
- **Accent Text:** All caps for emphasis
- **Logo Text:** CalorieToken wordmark (custom font from brand assets)

## Layout Principles (from examples)

1. **Clean Hierarchy:** Large logo/title ‚Üí key points ‚Üí website/CTA
2. **Icon-Driven:** Simple icons illustrate each benefit
3. **Color Blocking:** Distinct sections with solid colors (no complex gradients unless subtle)
4. **White Space:** Don't overcrowd; let elements breathe
5. **Consistent Branding:** Logo always visible, colors match palette
